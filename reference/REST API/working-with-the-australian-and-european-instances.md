If your organization is on the Australian or European instance, you will need to change the base URL from`https://api.fulcrumapp.com/api/v2/` to one of the following options:

AU: `https://api.fulcrumapp-au.com/api/v2/`

EU: `https://api.fulcrumapp-eu.com/api/v2/`
