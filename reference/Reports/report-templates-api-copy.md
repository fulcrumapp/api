Reports allow you to export your record data in easily-readable formats, like PDFs. You can read more about [creating custom PDF reports here.](https://help.fulcrumapp.com/en/articles/4208268-how-do-i-customize-a-pdf-report)

This endpoint can be helpful if you need to generate reports via the API.
