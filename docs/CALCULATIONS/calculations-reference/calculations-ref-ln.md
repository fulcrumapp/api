Returns the the logarithm of a number, base e (Euler's number).

# Parameters

`value` Number (**required**) - The value for which to calculate the logarithm, base `e`. Must be a positive number.

# Returns

Number

# Examples

```js
LN(5)

// returns 1.6094379124341003
```

```js
LN("Test")

// returns NaN
```
