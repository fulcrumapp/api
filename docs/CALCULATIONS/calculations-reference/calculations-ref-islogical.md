Checks whether a value is true or false.

# Parameters

`value` String (**required**) - The value to be verified as `true` or `false`.

# Returns

Boolean

# Examples

```js
ISLOGICAL(true)

// returns true
```

```js
ISLOGICAL("Text")

// returns false
```
