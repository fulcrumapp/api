Returns the string representation of a value. This function is useful for troubleshooting expressions.

# Parameters

`value` Number (__required__) - The value to inspect

# Returns

String

# Examples

```js
INSPECT([1, 2, 3])

// returns "[ 1, 2, 3 ]"
```
