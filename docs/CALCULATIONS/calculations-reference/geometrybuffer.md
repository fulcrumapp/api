Calculates a buffer for input features for a given radius. Units supported are miles, kilometers, and degrees.

# Parameters

`geometry` FeatureCollection | Geometry | Feature (**required**) - input geometry

`radius` number (**required**) - distance (in kilometers)

`options` object - options

# Returns

FeatureCollection | Geometry | Feature - buffered features

* **Note:** Leverages Turf.js ([https://turfjs.org](https://turfjs.org)) for calculations

# Examples

```js javascript
const point = {
  type: 'Point',
  coordinates: [-75.343, 39.984]
};

INSPECT(GEOMETRYBUFFER(point, 0.1))
```
