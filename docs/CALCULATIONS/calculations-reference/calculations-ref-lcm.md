Returns the least common multiple of one or more integers.

# Parameters

`var_args_values` Number (**required**) - Value(s) to consider to calculate the least common multiple.

# Returns

Number

# Examples

```js
LCM(2, 5)

// returns 10
```

```js
LCM(5, 22, 12)

// returns 660
```
