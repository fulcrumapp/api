Rounds a number down to the nearest integer that is less than or equal to it.

# Parameters

`value` Number (__required__) - Value to round down to the nearest integer.

# Returns

Number

# Examples

```js
INT(34.24)

// returns 34
```

```js
INT(-10.49)

// returns -11
```
