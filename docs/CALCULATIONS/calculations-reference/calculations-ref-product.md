Returns the result of multiplying a series of numbers together.

# Parameters

`var_args_values` Number (**required**) - Values to multiply together.

# Returns

Number

# Examples

```js
PRODUCT(4, 18, 3)

// returns 216
```

```js
PRODUCT(23, 5, 0)

// returns 0
```
