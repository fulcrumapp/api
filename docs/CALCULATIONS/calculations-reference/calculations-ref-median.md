Returns the median value in a set of numeric values.

# Parameters

`var_args_values` Number (**required**) - Values in which to find the median value.

# Returns

Number

# Examples

```js
MEDIAN(15,2,9,67,22)

// returns 15
```

```js
MEDIAN(1,10,4)

// returns 4
```
