Converts a provided year, month, and day into a date.

# Parameters

`year` Number (__required__) - year

`month` Number (__required__) - month

`day` Number (__required__) - day

# Returns

Date

# Examples

```js
DATE(2015, 12, 16)

// returns 2015-12-16
```
