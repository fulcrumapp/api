Converts any value to a number

# Parameters

`value` \* (**required**) - a value to convert to a number

# Returns

Number

# Examples

```js
NUM('12')

// returns 12
```

```js
NUM(12)

// returns 12
```

```js
NUM('a')

// returns NaN
```
