Calculates the square root of a given number.

# Parameters

`value` Number (**required**) - The number for which to calculate the square root.

# Returns

Number

# Examples

```js
SQRT(5)

// returns 2.23606797749979
```

```js
SQRT(-9)

// returns NaN
```
