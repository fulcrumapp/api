Returns the sign of a given input number. -1 if negative, 1 if positive, 0 if zero.

# Parameters

`value` Number (**required**) - The value to return the sign of.

# Returns

Number

# Examples

```js
SIGN(43)

// returns 1
```

```js
SIGN(-3)

// returns -1
```
