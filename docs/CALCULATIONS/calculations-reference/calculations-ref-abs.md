Returns the absolute value of a number

# Parameters

`value` Number (__required__) - The number of which to return the absolute value.

# Returns

Number - the absolute value of the `value` parameter

# Examples

```js
ABS(-1)

// returns 1
```


```js
ABS(42)

// returns 42
```
