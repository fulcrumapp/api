Returns the the logarithm of a number, base 10.

# Parameters

`value` Number (__required__) - The value for which to calculate the logarithm, base 10.

# Returns

Number

# Examples

```js
LOG10(100)

// returns 2
```

```js
LOG10(50)

// returns 1.6989700043360185
```
