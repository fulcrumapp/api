Checks whether the provided value is an even number.

# Parameters

`value` Number (__required__) - The value to be verified as even.

# Returns

Boolean

# Examples

```js
ISEVEN(4)

// returns true
```

```js
ISEVEN(5)

// returns false
```

```js
ISEVEN(4.12)

// returns true
```
